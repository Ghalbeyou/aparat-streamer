# Aparat Streamer
Stream local file videos to aparat servers!

## Requirements
- Python
- vidgear
- opencv2

## How to install
- Download files
- Open folder and install requirements: `pip install -r requirements.txt`
- Edit `config.json`
- Run `python3 main.py`
- Enjoy!
