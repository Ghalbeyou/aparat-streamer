Stream local file videos to aparat servers!

## Requirements
- Python
- vidgear
- opencv2

## How to install
- Download files
- Open folder and install requirements: `pip install -r requirements.txt`
- Edit `config.json`
- Run `python3 main.py`
- Enjoy!
## Features
- Stream with audio
- Quality is editable (in line `18`)
- Server is editable (in line `35`)
- Can stream 24/7
- Stream's local recorded file
